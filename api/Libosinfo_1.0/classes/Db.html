identify_media(media)¶
Parameters:media (Libosinfo.Media) – the installation media data
Returns:True if media was found in db, False otherwise
Return type:bool

Try to match a newly created media with a media description from db. If found, media will be filled with the corresponding information stored in db. In particular, after a call to Libosinfo.Db.identify_media (), if the media could be identified, its OsinfoEntify ::id and Libosinfo.Media ::os properties will be set.
