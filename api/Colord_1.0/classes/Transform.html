process(data_in, data_out, width, height, rowstride, cancellable)¶
Parameters:
  • data_in (object) – the data buffer to convert
  • data_out (object) – the data buffer to return, which can be the same as data_in
  • width (int) – the width of data_in
  • height (int) – the height of data_in
  • rowstride (int) – the rowstride of data_in, typically the same as width
  • cancellable (Gio.Cancellable or None) – A GLib.Error, or None
Raises:

GLib.GError
Returns:

True if the pixels were successfully transformed.
Return type:

bool

Processes a block of data through the transform. Once the transform has been setup it is cached and only re-created if any of the formats, input, output or abstract profiles are changed.
