Properties¶

Name Type Flags Description
clamp-value bool r/w Clamp the adjustment value between the lower and upper values, respecting the page-size.
elastic bool r/w Make interpolation behave in an ‘elastic’ way and stop clamping value.
lower float r/w Lower bound
page-increment float r/w Page increment
page-size float r/w Page size
step-increment float r/w Step increment
upper float r/w Upper bound
value float r/w Current value
