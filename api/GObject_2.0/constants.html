GObject.PARAM_MASK = 255¶

Mask containing the bits of GObject.ParamSpec.flags which are reserved for GLib.

GObject.PARAM_READWRITE = 0¶

GObject.ParamFlags value alias for GObject.ParamFlags.READABLE | GObject.ParamFlags.WRITABLE.

GObject.PARAM_STATIC_STRINGS = 0¶

GObject.ParamFlags value alias for GObject.ParamFlags.STATIC_NAME | GObject.ParamFlags.STATIC_NICK | GObject.ParamFlags.STATIC_BLURB.

Since:2.13.0
GObject.PARAM_USER_SHIFT = 8¶

Minimum shift count to be used for user defined flags, to be stored in GObject.ParamSpec.flags. The maximum allowed is 30 + GObject.PARAM_USER_SHIFT.
