get_font_name()¶
Returns:A string with the name of the current font, or None if no font is selected. You must free this string with GLib.free ().
Return type:str

Gets the currently-selected font name.

Note that this can be a different string than what you set with Gtk.FontSelectionDialog.set_font_name (), as the font selection widget may normalize font names and thus return a string with a different structure. For example, “Helvetica Italic Bold 12” could be normalized to “Helvetica Bold Italic 12”. Use Pango.FontDescription.equal () if you want to compare two font descriptions.
