Signals¶

Name Parameters Return Description
color-set     The ::color-set signal is emitted when the user selects a color. When handling this signal, use Gtk.ColorButton.get_rgba () to find out which color was just selected. Note that this signal is only emitted when the user changes the color. If you need to react to programmatic color changes as well, use the notify ::color signal.

Details¶

class Gtk.ColorButton(label=None, stock=None, use_stock=False, use_underline=False, **kwds)¶

Bases: Gtk.Button, Gtk.ColorChooser

The Gtk.ColorButton is a button which displays the currently selected color an allows to open a color selection dialog to change the color. It is suitable widget for selecting a color in a preference dialog.

static new()¶
Returns:a new color button
Return type:Gtk.Widget

Creates a new color button.

This returns a widget in the form of a small button containing a swatch representing the current selected color. When the button is clicked, a color-selection dialog will open, allowing the user to select a color. The swatch will be updated to reflect the new color when the user finishes.
