GnomeKeyring.item_create_sync(keyring, type, display_name, attributes, secret, update_if_exists)¶
Parameters:
  • keyring (str or None) – The name of the keyring in which to create the item, or None for the default keyring.
  • type (GnomeKeyring.ItemType) – The item type.
  • display_name (str) – The name of the item. This will be displayed to the user where necessary.
  • attributes (GLib.Array) – A (possibly empty) list of attributes to store with the item.
  • secret (str) – The password or secret of the item.
  • update_if_exists (bool) – If true, then another item matching the type, and attributes will be updated instead of creating a new item.
Returns:

GnomeKeyring.Result.OK if the operation was succcessful or an error result otherwise.
Return type:

GnomeKeyring.Result, item_id: int

Create a new item in a keyring.

The secret must be a null terminated string. It should be allocated using secure memory whenever possible. See GnomeKeyring.memory_strdup ()

The user may have been prompted to unlock necessary keyrings. If None is specified as the keyring and no default keyring exists, the user will be prompted to create a new keyring.

When update_if_exists is set to True, the user may be prompted for access to the previously existing item.

For an asynchronous version of this function see GnomeKeyring.item_create ().

GnomeKeyring.item_get_info_full_sync(keyring, id, flags)¶
Parameters:
  • keyring (str or None) – The name of the keyring in which the item exists, or None for the default keyring.
  • id (int) – The id of the item
  • flags (int) – The parts of the item to retrieve.
Returns:

GnomeKeyring.Result.OK if the operation was succcessful or an error result otherwise.
Return type:

GnomeKeyring.Result, info: GnomeKeyring.ItemInfo

Get information about an item, optionally retrieving its secret.

If flags includes GnomeKeyring.ItemInfoFlags.SECRET then the user may be prompted if the calling application doesn’t have necessary access to read the item with its secret.

A GnomeKeyring.ItemInfo structure will be returned in info. Certain fields of this structure may be None or zero if they were not specified in flags. This must be freed using GnomeKeyring.ItemInfo.free ().

For an asynchronous version of this function see GnomeKeyring.item_get_info_full ().
