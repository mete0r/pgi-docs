Details¶

class GstBase.DataQueue(**kwargs)¶

Bases: GObject.Object

GstBase.DataQueue is an object that handles threadsafe queueing of objects. It also provides size-related functionality. This object should be used for any Gst.Element that wishes to provide some sort of queueing functionality.

Parameters:type (GObject.GType) – The GLib.Type of the item to drop.
Returns:True if an element was removed.
Return type:bool

Pop and unref the head-most Gst.MiniObject with the given GLib.Type.

flush()¶

Flushes all the contents of the queue. Any call to GstBase.DataQueue.push and GstBase.DataQueue.pop will be released. MT safe.

Inform the queue that the limits for the fullness check have changed and that any blocking GstBase.DataQueue.push () should be unblocked to recheck the limts.

peek(item)¶
Parameters:item (GstBase.DataQueueItem) – pointer to store the returned GstBase.DataQueueItem.
Returns:True if an item was successfully retrieved from the queue.
Return type:bool

Retrieves the first item available on the queue without removing it. If the queue is currently empty, the call will block until at least one item is available, OR the queue is set to the flushing state. MT safe.

Retrieves the first item available on the queue. If the queue is currently empty, the call will block until at least one item is available, OR the queue is set to the flushing state. MT safe.

push(item)¶
Parameters:item (GstBase.DataQueueItem) – a GstBase.DataQueueItem.
Returns:True if the item was successfully pushed on the queue.
Return type:bool

Pushes a GstBase.DataQueueItem (or a structure that begins with the same fields) on the queue. If the queue is full, the call will block until space is available, OR the queue is set to flushing state. MT safe.

Note that this function has slightly different semantics than Gst.Pad.push () and Gst.Pad.push_event (): this function only takes ownership of item and the Gst.MiniObject contained in item if the push was successful. If False is returned, the caller is responsible for freeing item and its contents.

push_force(item)¶
Parameters:item (GstBase.DataQueueItem) – a GstBase.DataQueueItem.
Returns:True if the item was successfully pushed on the queue.
Return type:bool

Pushes a GstBase.DataQueueItem (or a structure that begins with the same fields) on the queue. It ignores if the queue is full or not and forces the item to be pushed anyway. MT safe.

Note that this function has slightly different semantics than Gst.Pad.push () and Gst.Pad.push_event (): this function only takes ownership of item and the Gst.MiniObject contained in item if the push was successful. If False is returned, the caller is responsible for freeing item and its contents.

set_flushing(flushing)¶
Parameters:flushing (bool) – a bool stating if the queue will be flushing or not.

Sets the queue to flushing state if flushing is True. If set to flushing state, any incoming data on the queue will be discarded. Any call currently blocking on GstBase.DataQueue.push or GstBase.DataQueue.pop will return straight away with a return value of False. While the queue is in flushing state, all calls to those two functions will return False.

MT Safe.
