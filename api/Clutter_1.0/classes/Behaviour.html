class Clutter.Behaviour(**kwargs)¶

Bases: GObject.Object, Clutter.Scriptable

Clutter.Behaviour-struct contains only private data and should be accessed with the functions below.

actors_foreach(func, *data)¶
Parameters:
  • func (Clutter.BehaviourForeachFunc) – a function called for each actor
  • data (object) – optional data to be passed to the function, or None

Calls func for every actor driven by behave.

apply(actor)¶
Parameters:actor (Clutter.Actor) – a Clutter.Actor

Applies behave to actor. This function adds a reference on the actor.

get_actors()¶
Returns:a list of actors. You should free the returned list with GLib.SList.free () when finished using it.
Return type:[Clutter.Actor]

Retrieves all the actors to which behave applies. It is not recommended for derived classes to use this in there alpha notify method but use Clutter.Behaviour.actors_foreach as it avoids alot of needless allocations.

get_alpha()¶
Returns:a Clutter.Alpha object, or None if no alpha object has been bound to this behaviour.
Return type:Clutter.Alpha

Retrieves the Clutter.Alpha object bound to behave.

get_n_actors()¶

Return type:int

Gets the number of actors this behaviour is applied too.

Parameters:actor (Clutter.Actor) – a Clutter.Actor

Check if behave applied to actor.

remove(actor)¶
Parameters:actor (Clutter.Actor) – a Clutter.Actor

Removes actor from the list of Clutter.Actor s to which behave applies. This function removes a reference on the actor.

remove_all()¶

Removes every actor from the list that behave holds.

set_alpha(alpha)¶
Parameters:alpha (Clutter.Alpha) – a Clutter.Alpha or None to unset a previously set alpha

Binds alpha to a Clutter.Behaviour. The Clutter.Alpha object is what makes a behaviour work: for each tick of the timeline used by Clutter.Alpha a new value of the alpha parameter is computed by the alpha function; the value should be used by the Clutter.Behaviour to update one or more properties of the actors to which the behaviour applies.

If alpha is not None, the Clutter.Behaviour will take ownership of the Clutter.Alpha instance.
