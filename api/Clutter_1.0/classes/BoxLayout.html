set_use_animations(animate)¶
Parameters:animate (bool) – True if the layout should use animations

Sets whether layout should animate changes in the layout properties

The duration of the animations is controlled by Clutter.BoxLayout.set_easing_duration (); the easing mode to be used by the animations is controlled by Clutter.BoxLayout.set_easing_mode ().

Enabling animations will override the easing state of each child of the actor using layout, and will use the Clutter.BoxLayout :easing-mode and Clutter.BoxLayout :easing-duration properties instead.

Parameters:vertical (bool) – True if the layout should be vertical

Sets whether layout should arrange its children vertically alongside the Y axis, instead of horizontally alongside the X axis
